# Zegowska Szama
jest to rewelacyjna aplkikacja pozwalająca ucznią widzieć co jest dostępne w sklepiku oraz zamawiać jedzenie online. Aplikacja posiada rewolucjonalny interfejs logowania z bajecznym tłem oraz z nowoczesnym systemem rejestracji.

Nasi programiści ciągle pracują nad ulepszeniami Zegowskiej Szamy, aby wasze buły zawsze były ciepłe. A mrożona kawa zimna. Z niewielką pomocą sztucznej inteligencji nasz zespół był w stanie osiągnąc sukces graficzny aplikacji. Z podziękowaniami dla Pana Grzegorza Szymkowiaka (bez niego nigdy byśmy nie zdołali tego osiągnąć).
